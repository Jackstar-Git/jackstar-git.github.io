Text Editor

This simple but still very useful Text-Editor program is inspired by the windows notepad. It has many different features, like saving files, loading files and even a custom file type, that is still work in progress. It also has the following other features:

  • Generate placeholder text ("lorem impsum")
  • Insert the current time and date with one simple keyboard shortcut
  • Simple file encryption
  • Change font size, style and color
  • Supports zooming in and out
  • Has features like count words and symbols
  • Function to easily replace text acrosss the etire file
  • Easy text highlighting for important things
  • much more...


If you want to check out the program or you are interested in downloading it, click here: Text Editor Download!
