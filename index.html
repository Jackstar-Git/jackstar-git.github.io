About Me

Hey, I'm Jackstar, a Python programmer from the heart of Europe. Most of my free time is spent immersed in the fascinating world of coding, a journey that's been going strong for nearly 3 years now and there is no end in near sight.

The thing I like most about coding is, that I experience something new every day. No project is equal to another one, so creativity is very important. Another thing I like about programming is the fact, that you are able to create (almost) everything you want on your own. You jsut have to try hard and stay onto it.
My other hobbies besides coding include also gaming. There are two generes of games I really enjoy playing: Racing- and Storygames.

But this website isn't just about code - it's a space where you can get to know me better. Welcome to my digital corner, designed for everyone to get a closer look into who I am and what I do!

July 2022

In July 2022, I started my coding journey with Python, driven by the idea that I just wanted to create a custom bot for my Discord Server. What began as a casual interest swiftly involved into a real "full-time hobby". Over the years I always kept on learning new things and expanded my field of knowledge.

October 2022

When I finished my first program in October 2022 I still had almost zero knowledge about coding. But I kept on going and I never stopped learning.

Janurary 2023

Around this time I had learned HTML and CSS. This was also the time I created my first Discord bot.

Overall, this was the main reason I started coding in the first place.

August 2023

In August of 2023 I started learning how to use the MongoDB Database. This was also the time when I started to learn the basics of JavaScript to extend the useage range of my HTML and CSS Skills.

Later this month was also the time I started and completed my first paid job as a Python developer.

Feburary 2024

From August 2023 onwards I did more and more projects, paid and for free. Since every project was completyl different from another, I had to study more things.

I also began working on my first 2D-Game as my side-project. Now let's see what the future brings...

Text Editor

This simple but still very useful Text-Editor program is inspired by the windows notepad. It has many different features, like saving files, loading files and even a custom file type, that is still work in progress. It also has the following other features:

  • Generate placeholder text ("lorem impsum")
  • Insert the current time and date with one simple keyboard shortcut
  • Simple file encryption
  • Change font size, style and color
  • Supports zooming in and out
  • Has features like count words and symbols
  • Function to easily replace text acrosss the etire file
  • Easy text highlighting for important things
  • much more...
